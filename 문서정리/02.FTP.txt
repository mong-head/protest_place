<FTP>

파일 전송하기 위한 통신규약

ex ) http -> http://url : url로 데이터 가지고 오는데 http프로토콜을 사용하겠다는 말

ex ) ftp -> ftp://url 도 비슷한 거임

http : 웹서버, 웹브라우저 
FTP : FTP 서버, FTP 클라이언트

FTP 클라이언트가 FTP서버와 교신해서 서버쪽 파일에 업로드, 다운로드 가능


<FTP 클라이언트>

여러가지 있지만, 그 중 filezilla사용

filezilla client 다운로드받기

*화면 설명
 - 로컬 사이트 : 내 컴퓨터 자체
 - 리모트 사이트 : 내가 접근하고자 하는 서버, 원격지에 있는 컴퓨터

 -파일 밑에 있는 아이콘 눌러서 new site추가
 -호스트에 도메인적기(원격하고자 하는거)
 -protocol : 요즘은 FTP는 보안상문제로 쓰지 않고, SFTP사용
  **하지만 나는 무료 호스팅을 사용해서 SFTP는 안돼고, FTP로만 됨
